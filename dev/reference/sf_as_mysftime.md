# Convert an `sf` to an `sftime`

Convert an `sf` object to an `sftime` object. `x` must contain a
time-defining column, identified in `timename`.

## Usage

``` r
sf_as_mysftime(x, timename)
```

## Arguments

- x:

  an `sf` object

- timename:

  character: name of time column in x

## Value

an `sftime` object
