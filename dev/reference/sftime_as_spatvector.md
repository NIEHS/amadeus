# Convert an `sftime` to a `SpatVector`

Convert an `sftime` object to a `SpatVector` object.

## Usage

``` r
sftime_as_spatvector(x)
```

## Arguments

- x:

  an `sftime` object

## Value

a `SpatVector` object
