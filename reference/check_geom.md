# Check that `geom` value is one of `FALSE`, `"sf"`, or `"terra"`

Check that `geom` value is one of `FALSE`, `"sf"`, or `"terra"`.

## Usage

``` r
check_geom(geom)
```

## Arguments

- geom:

  FALSE/"sf"/"terra".'

## Value

NULL; will stop if `geom` is not one of the three options
